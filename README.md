login-logout form using java.
This project demonstrates a basic user registration and login system using Java, JSP, Servlets, and MySQL. It allows new users to register with their details and existing users to log in using their credentials. The application validates input, stores data securely in a database, and provides a simple user interface for interaction.
Features:
User Registration with validation
Secure Login Authentication
Password storage using database
Error handling for invalid login
Simple and clean UI
Database connectivity using JDBC

 
 ## Technologies Used
- Java 17
- JSP / Servlets
- JDBC
- MySQL Database
- Apache Tomcat 10
- HTML
